Sobre a Escola

O Colégio Estadual Governador Adolpho de Oliveira Franco está localizado em Astorga, Paraná. A escola oferece ensino de qualidade e promove a formação integral dos alunos.

Diretor(a): Marli de Marco

O sistema de consulta de escolas do estado do Paraná confirma o registro da escola em Astorga.

Curso voltado para agricultura, agropecuária e tecnologias do campo.

Sobre o Curso Técnico

Objetivo

Preparar o aluno para atuar no mercado de tecnologia com conhecimentos práticos e teóricos em desenvolvimento de sistemas, banco de dados, programação mobile e ciência de dados.

Matérias

  • Programação Mobile
  • Banco de Dados
  • Computação Gráfica
  • Análise e Projeto de Sistema
  • Programação Back End
  • Ciência de Dados
  • Programação e Desenvolvimento de Sistema

Carga Horária

O curso possui duração de 1800 horas distribuídas em 3 anos, com aulas teóricas e práticas.

Habilidades Desenvolvidas

  • Desenvolvimento de aplicativos móveis
  • Criação e manutenção de bancos de dados
  • Modelagem de sistemas e projetos de software
  • Programação back end e front end
  • Análise e interpretação de dados
